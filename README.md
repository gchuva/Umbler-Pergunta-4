# Umbler-Pergunta-4

# Qual a vantagem de usar containers em vez de configurar tudo manualmente no servidor?
R: Conseguimos padronizar, agilizar e flexibilizar configurações dentro de múltiplos containeres ao mesmo tempo, ou seja se funcionar em 1 funcionará em outro.

# O que significa idempotência nesse contexto? O que acontece se duas vezes num ambiente já configurado?
R: Lhe permite executar uma operação inúmeras vezes obtendo sempre o mesmo resultado.
